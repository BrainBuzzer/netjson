NetJSON: data interchange format for networks

Specification (HTML) Specification (ePub) Github Issue tracker FAQs Implementations Mailing list

NetJSON is a data interchange format based on JavaScript Object Notation (JSON) designed to describe the basic building blocks of layer2 and layer3 networks.

It defines several types of JSON objects and the manner in which they are combined to represent a network: configuration of devices, monitoring data, network topology and routing information.
